# BrainAI_Diane_Plugins
## Module und Plugins für Diane

*Diane* ist eine deterministische XAGI, die via Plugins und Module erweitert werden kann.
In diesem Repository sind nur ihre Module und Plugins vorhanden. Diane selber (der Core) ist _nicht_ Teil dieses Repositories.
